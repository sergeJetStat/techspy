How it works

Four steps from URL to leads

A fully automated pipeline — just add domains and describe what you're looking for.

01

Crawl & extract

Async HTTP fetcher downloads pages, capturing headers, HTML, cookies, and script paths at up to 50 concurrent requests.

02

Fingerprint matching

Deterministic engine scores each detected pattern. Confidence ≥60 triggers a detection. Version strings are extracted where available.

03

AI enrichment

Claude-powered agents research unknown signals and propose new fingerprints. Each agent works autonomously using tool-calling loops.

04

Natural language leads

Query in plain English. The Lead Agent filters the database with AND / OR / NOT logic and exports CSV or JSON for your CRM.
